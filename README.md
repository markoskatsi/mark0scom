# mark0s.com

A personal portfolio website showcasing my projects, experience, and a bit about me.

## Features

- **Projects** — a curated list of things I've built
- **Experience** — my professional background
- **About** — a short intro
- **Get CV** — request my CV via email, sent through a FastAPI endpoint
- **Contact form** — send me a message, also powered by a FastAPI backend

## Tech Stack

- React + TypeScript + Vite
- FastAPI (CV request + contact form)
- Self-hosted on a home server, exposed via Cloudflare Tunnel
